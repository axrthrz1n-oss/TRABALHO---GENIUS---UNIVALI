# TRABALHO---GENIUS---UNIVALI
GENIUS - O Jogo da Memória Desenvolvido como trabalho da disciplina de Algoritmos e Programação, este jogo é uma recriação do clássico Genius, onde o jogador deve memorizar e repetir sequências de números que ficam progressivamente mais longas e desafiadoras. Com três níveis de dificuldade, o jogo testa os limites da sua memória e atenção
